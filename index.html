Selectores Cheatsheet

Ideas iniciales

CCS puede seleccionar elementos HTML por tipo, clase, ID o atributo y además, por el selector universal.

La especifidad es el orden por el cual el navegador decide que estilos CSS van a ser aplicados a un elemento concreto. Podemos hacer un cálculo para anticipar la decisión en función del valor que representa cada selector.

Una buena práctica en CSS es dar estilos usando el menor nivel de especifidad necesario, de manera que pueda ser re-estilado de manera sencilla.

Lo mejor es empezar a dar estilos por tipos. Si no es posible, usa un selector de clase y si eso no es suficiente, usa el ID.

Tabla de Especifidad

Orden de especifidad Valor Selector Objetivo Snippet
1 1000 Atributo Style inline (<style>) El propio elemento HTML que contiene el atributo style <h1 style="color: #ffffff">Heading</h1>
2 100 ID (#) Un elemento con ID #titulo-principal { font-family: cursive; }
3 10 Clases (.) Elementos con dicha clase .título { color: teal; }
4 10 Atributo (href, src...) Elementos que contienen el atributo dado [href] { color: lightblue; }
5 1 Tipos (a, p, header...) Elementos de dicho tipo p { color: green; }
6 0 Universal (*) Todos los elementos, clases, tipos... * { font-family: Verdana; }

Notas

Un elemento HTML puede tener varias clases. Varios selectores de clases puede apuntar a un mismo elemento.

Las pseudo-clases pueden asignarse a cualquier selector. Se escriben con ':' seguido de un nombre. Ejemplo: p:hover { color: darkorange; }

Se pueden combinar selectores para dar estilos a elementos que cumplan varias condiciones. Ejemplo: h2.special {color: darkorange; } dará el estilo a los h2 que tengan la clase 'special'

A igual nivel o valor de especifidad, aplicará el estilo que más abajo haya sido escrito en el código.
